# 方案13：标准模型户（一户多贷路径）v1.0

> ⚠️ **声明**：本文为内部筹划/测算用，不对外展示，不构成任何放贷承诺或合规意见。
> 
> 我们不做多头欺诈授信和违法套利，只做基于真实经营的结构优化。

**最后更新日期**：2025-12-20  
**关联业务**：H业务（开票）+ 流水优化 + 资质认定（综合）  
**方案分类**：综合方案  
**信息来源**：综合服务商案例  
**成功率分级**：高（80%+）  
**适用地区**：全国

---

## 一、方案概述

### 适用场景
- **条件缺口**：企业需要综合提升体质，实现一户多贷路径
- **目标产品**：发票贷 + 流水贷 + 科创贷（一户多贷组合）
- **典型客户**：
  - 有一定经营基础，但单维度不足的企业
  - 需要大额资金，单一产品无法满足的企业
  - 愿意投入6-9个月进行综合改造的企业

### 改造目标
设计"标准模型户"画像，实现：
- 月开票500万（年6000万）→ 票贷额度300-500万
- 月流水5000万（年6亿）→ 流水贷额度300-500万
- 获得科小/创新型/高新资质 → 科创贷额度200-500万
- **综合总额度：1200-1500万**（低息部分800-1000万）

### 核心逻辑
通过综合改造票税体质、流水体质、资质体质，构建"标准模型户"，实现一户多贷的合规路径。

## 三、执行步骤

### Step 1：打底阶段（1-2个月）

**动作**：
- 梳理真实贸易逻辑，确保流水、发票和业务故事统一
- 设计资金流路径：保证每日有留存、有结息
- 补齐社保1-3人
- 修正明显硬伤（如法人征信瑕疵）

**周期**：1-2个月  
**成本**：约2-3万（社保、基础优化）  
**产出**：基础体质达标，准备进入改造期

---

### Step 2：发票/流水+科小（3-4个月）

**动作**：
- 对齐目标开票结构（500万/月）- 通过H业务实现
- 对齐资金流水结构（5000万/月）- 流水优化
- 启动科小认定：准备软著、研发费用证明、人员结构

**周期**：2个月（与开票同步进行）  
**成本**：
- H业务开票：约6万（500万×1.2%×10个月）
- 税费：约4万
- 科小认定：1-3万
- **小计：约11-13万**

**产出**：
- 连续开票记录（6个月以上）
- 科小资质认定
- 流水结构优化完成

---

### Step 3：创新型+一户多贷初步落地（5-6个月）

**动作**：
- 补一件与主营强相关的专利 → 申报创新型中小企业
- 正式在2-3家银行落地票贷/经营贷，形成首轮授信
- 观察实际利率区间，调整后续组合

**周期**：2个月  
**成本**：
- 创新型认定：1-2万
- 贷款申请服务费：0-2万（可选）
- **小计：约1-4万**

**产出**：
- 创新型资质认定
- **首批贷款到账：约800-1000万**

---

### Step 4：高新+额度优化（7-9个月，可选）

**动作**：
- 若企业本身研发基础尚可，再考虑高新认定
- 在已有授信基础上，做结构优化：
  - 尽量用低息政策类贷款替代部分高息短期周转
  - 控制总负债与现金流比，避免"多头授信"风险

**周期**：3-6个月（高新认定周期）  
**成本**：
- 高新认定：3-8万（综合成本）
- **小计：约3-8万**

**产出**：
- 高新资质认定（可选）
- 综合额度提升至1200-1500万

## 六、效果预测

### 改造前后对比

| 维度 | 改造前 | 改造后 | 提升 |
|------|--------|--------|------|
| 年开票额 | 100-500万 | 6000万 | +5500-5900万 |
| 年流水 | 5000万-2亿 | 6.6亿 | +4.6-6.1亿 |
| 资质 | 无 | 科小+创新型（可选高新） | 从无到有 |
| 可贷额度 | 50-200万 | 1200-1500万 | +1000-1450万 |
| 利率区间 | 8%-15% | 3%-12%（加权） | 更优惠 |

### ROI计算

```
投入成本：18-28万（含高新）
预计增加贷款额度：1000-1450万
预计节省利息：50-100万/年（按利率差5%计算）

方式1：按额度计算
ROI = (1000万额度 / 20万投入) = 5000%

方式2：按利息节省计算
年节省利息：1000万 × 5% = 50万
2年节省：100万
ROI = (100万收益 / 20万投入) × 100% = 500%
回本周期：约0.4年（5个月）

综合ROI：500%-5000%（取决于计算方式）
```

## 七、风险提示

### ⚠️ 风险1：多头授信风险
- **描述**：央行征信会记录查询次数、授信额度、用信情况，短期内多家银行频繁查询+快速加杠杆，容易被风控打标
- **概率**：中
- **影响**：额度可能被压缩/撤销
- **应对**：少量银行、稳定合作、避免极端加杠杆，每家银行都如实披露其他授信情况

### ⚠️ 风险2：成本超预期
- **描述**：资质认定综合成本可能超出预算
- **概率**：中
- **影响**：总成本可能增加5-10万
- **应对**：提前确认各项费用，分阶段投入

### ⚠️ 风险3：周期延长
- **描述**：银行审批或资质认定周期可能延长
- **概率**：中
- **影响**：总周期可能延长至10-12个月
- **应对**：提前规划，预留缓冲时间

### ⚠️ 风险4：合规风险
- **描述**：如果操作不规范，可能引起税务或银行关注
- **概率**：低（合规操作下）
- **影响**：税务稽查、贷款被拒
- **应对**：严格按照"四个真实"操作（真实货物、真实发票、真实交易、真实仓单）

## 八、关键前提与合规原则

### 🔴 必须遵守的原则

| 原则 | 说明 |
|------|------|
| **真实经营** | 开票、流水、社保、资质都要有真实支撑，严禁空转和虚开 |
| **如实披露** | 向银行如实披露已有负债和授信情况，不提供虚假材料 |
| **合规优先** | 不刻意规避"多头授信"监管，不做极端加杠杆 |

### ✅ 安全做法

> 不是"教人怎么钻空子"，而是：
> - 设计**额度+还款+现金流**整体结构
> - **少量银行、稳定合作、避免极端加杠杆**
> - 每家银行都如实披露其他授信情况

## 十一、话术要点

### 客户痛点挖掘
```
"老板，您现在贷款额度是多少？利率多少？"
"您企业一年开票多少？流水多少？"
"您知道为什么额度上不去吗？其实银行主要看开票、流水、资质..."
```

### 方案介绍
```
"根据您的情况，我建议通过我们的标准模型户方案来综合提升体质。"
"简单说，就是通过开票提升、流水优化、资质认定三个维度，帮您构建一个'标准体质'。"
"大概需要6-9个月，投入约18-28万，预计能把综合额度做到1200-1500万。"
```

### 异议处理
```
Q：28万太贵了
A："我们来算一笔账：投入28万，能多贷1000万（从200万到1200万），相当于用28万撬动1000万资金，而且每年还能省利息50-100万。您觉得划算吗？"

Q：要6-9个月太慢了
A："这是综合改造的合理周期。好消息是，一旦建立起来，以后就是您的长期资产了。而且首批贷款在6个月左右就能到账。"

Q：能保证贷到吗？
A："我们不承诺具体结果，但根据标准模型户的设计，只要按方案执行到位，成功率在90%以上。而且我们会帮您同时准备2-3家银行，确保成功率。"
```
