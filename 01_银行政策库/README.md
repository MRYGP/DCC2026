### ⚠️ 重要说明

**数据状态**：`待验证`

这些资料是从外部收集的原始数据，存在以下问题：
- ❓ 信息可能过时（银行产品政策经常变）
- ❓ 部分字段可能不准确
- ❓ 需要通过电话/实际申请验证

**使用原则**：
1. 可作为**参考**，不能直接作为**结论**
2. 录入产品总表前，必须标注`待验证`
3. 验证后更新状态为`已验证`并标注验证日期

## 验证流程

1. 从 `daikuanziliao` 提取产品信息
2. 录入产品总表，标注 `状态: 待验证`
3. 通过以下方式验证：
   - 银行官网查询
   - 致电银行客户经理
   - 实际申请测试
4. 验证后更新状态为 `已验证`，标注验证日期和验证人
