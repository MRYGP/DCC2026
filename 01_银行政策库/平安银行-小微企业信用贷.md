## 二、准入条件

### 硬性条件（必须满足）

**企业条件**：
- [ ] 企业成立时间：≥1年
- [ ] 企业类型：小微企业、个体工商户
- [ ] 年营业收入：建议≥100万
- [ ] 经营状态：正常经营，未列入异常名录
- [ ] 有稳定的经营数据（流水/开票/纳税等）
- [ ] 行业限制：不支持房地产、金融、娱乐等高风险行业

**法人条件**：
- [ ] 年龄：22-60周岁
- [ ] 征信良好，无重大不良记录
- [ ] 实际控制人承担连带责任担保

**数据要求（满足其一即可）**：
- [ ] 有银行流水（企业或个人经营流水）
- [ ] 有开票记录（年开票≥50万）
- [ ] 有纳税记录（年纳税≥5000元）
- [ ] 有房产可抵押
- [ ] 有保单、公积金等资产证明

### 加分条件（提升额度/降低利率）

**企业资质**：
- [ ] 高新技术企业：+额度，-利率
- [ ] 专精特新企业：+额度，-利率
- [ ] 平安银行存量客户：+通过率，+额度

**资产情况**：
- [ ] 有房产抵押：额度大幅提升，利率降低
- [ ] 有大额保单（年缴≥2万）：+额度
- [ ] 有公积金缴纳（月缴≥2000）：+额度

**征信情况**：
- [ ] 有其他银行贷款且还款正常：+通过率
- [ ] 个人信用卡使用良好：+通过率
- [ ] 征信查询次数少：+通过率

**经营数据**：
- [ ] 年营收稳定增长：+额度
- [ ] 有稳定的客户和订单：+通过率
- [ ] 开票/纳税连续性好：+额度

### 减分/排斥条件

**企业排斥**：
- [ ] 房地产开发、中介：直接拒绝
- [ ] 金融类企业：直接拒绝
- [ ] 娱乐、高污染行业：直接拒绝
- [ ] 经营异常、被执行：直接拒绝

**征信排斥**：
- [ ] 当前有逾期：直接拒绝
- [ ] 近2年内连续逾期3期或累计逾期6期：直接拒绝
- [ ] 有呆账、代偿、诉讼记录：直接拒绝
- [ ] 法人个人负债过高（资产负债率>80%）：拒绝

**经营数据排斥**：
- [ ] 无任何经营数据支撑：拒绝
- [ ] 流水、开票、纳税严重不符：拒绝
- [ ] 经营波动过大、异常：降额或拒绝

## 三、额度算法

### 基础额度计算

平安银行小微企业信用贷采用**综合评分+多产品体系**，根据企业情况匹配不同产品：

```
产品体系：
1. 纯信用贷：最高200万（基于流水/开票/纳税）
2. 保单贷：最高50万（基于商业保单）
3. 公积金贷：最高50万（基于公积金缴纳）
4. 抵押贷：最高3000万（基于房产抵押）

综合评分维度：
- 企业基本情况（20%）：成立年限、行业、规模
- 经营数据（30%）：流水、开票、纳税
- 征信情况（25%）：企业征信、法人征信
- 资产情况（15%）：房产、保单、公积金
- 平安关系（10%）：是否存量客户、存款等
```

### 影响因素

| 因素 | 影响 | 说明 |
|------|------|------|
| 有房产抵押 | 额度×3-10倍 | 抵押贷最高3000万 |
| 平安存量客户 | +15%-25% | 有贷款或存款记录 |
| 年营收500万以上 | +10%-20% | 经营规模加分 |
| 高新/专精特新资质 | +15%-25% | 政策支持类企业 |
| 有大额保单（年缴≥5万） | +20-50万 | 叠加保单贷产品 |
| 有公积金（月缴≥3000） | +20-50万 | 叠加公积金贷产品 |
| 征信查询次数多（近3月>8次） | -15%-25% | 风险扣分 |
| 个人负债率高（>70%） | -10%-20% | 还款能力扣分 |
| 流水波动大 | -5%-15% | 稳定性扣分 |

### 额度区间

**纯信用贷**：
- 起贷金额：5万元
- 典型额度：20-100万元
- 最高额度：200万元

**抵押贷**：
- 起贷金额：50万元
- 典型额度：200-1000万元
- 最高额度：3000万元

### 额度测算规则

**年营收与额度对应关系**（参考，纯信用）：

| 年营收 | 预估额度范围 | 说明 |
|--------|-------------|------|
| 100-300万 | 20-60万 | 小微企业 |
| 300-500万 | 60-100万 | 成长型企业 |
| 500-1000万 | 100-150万 | 中型企业 |
| 1000-3000万 | 150-200万 | 较大企业 |

**房产抵押与额度对应关系**：

| 房产评估值 | 抵押率 | 预估额度 | 说明 |
|-----------|--------|---------|------|
| 200-500万 | 70% | 140-350万 | 住宅类房产 |
| 500-1000万 | 70% | 350-700万 | 优质住宅 |
| 1000-3000万 | 65% | 650-1950万 | 大额抵押 |
| 3000万以上 | 60% | 1800-3000万 | 最高额度 |

*以上数据为示例，实际以银行审批为准*

## 五、申请流程

### 申请方式

**方式1：线上申请**
- 通过平安银行官网、APP或小程序申请
- 在线填写信息、上传材料
- 系统自动审批或转人工

**方式2：线下申请**
- 联系平安银行客户经理
- 到网点面签
- 提交材料

### 详细流程

**Step 1：咨询与评估（30分钟）**
- 联系平安银行客户经理或在线咨询
- 初步评估企业情况
- 确定适合的产品方案

**Step 2：提交申请（1小时）**
- 填写申请表
- 上传营业执照、身份证等基础材料
- 授权采集经营数据

**Step 3：审批调查（1-3天）**
- 平安银行初审材料
- 查询征信
- 核实经营数据
- 评估房产（如有抵押）

**Step 4：审批决策（1-2天）**
- 分行审批
- 确定额度和利率
- 短信通知审批结果

**Step 5：签约放款（1天）**
- 线上或线下签约
- 办理抵押登记（如有）
- 资金发放到指定账户

### 总耗时

- 纯信用贷：最快1-3天，一般3-5天
- 抵押贷：最快5天，一般7-15天（含抵押登记）

## 六、费用说明

### 利息费用

**年化利率**：3.51%-8%（根据产品类型和客户资质）

**利率定价参考**：

**纯信用贷**：
- 优质客户（高新+平安存量）：4%-5%
- 良好客户（数据好+征信好）：5%-6.5%
- 一般客户：6.5%-8%

**抵押贷**：
- 优质房产+优质客户：3.51%-4%
- 一般房产+良好客户：4%-5%
- 普通情况：5%-6%

**示例1（纯信用）**：
```
贷款金额：100万
期限：12个月
年化利率：6%
还款方式：等额本息

月还款：100万 × [0.5% × (1+0.5%)^12] ÷ [(1+0.5%)^12 - 1]
      ≈ 86,066元/月
总利息：约3.3万元
```

**示例2（抵押贷）**：
```
贷款金额：300万
期限：36个月
年化利率：4%
还款方式：先息后本

月利息：300万 × 4% ÷ 12 = 10,000元/月
总利息：10,000元 × 36 = 36万元
到期还本：300万元
```

### 其他费用

**纯信用贷**：
- 无担保费
- 无手续费
- 无提前还款违约金

**抵押贷**：
- 房产评估费：约0.1%-0.5%（按评估值）
- 抵押登记费：约数百元
- 保险费：约0.05%-0.1%/年（可选）

### 综合成本

**纯信用贷综合成本 = 利息（4%-8%）**

**抵押贷综合成本 = 利息（3.51%-6%）+ 评估费等（首次约0.3%-0.8%）**

## 七、备注

### 产品优势

**1. 产品线丰富、灵活匹配**
- 有纯信用贷、抵押贷、保单贷、公积金贷等多种产品
- 根据企业情况灵活匹配最适合的方案
- 可多产品组合，叠加额度

**2. 抵押贷额度高、利率低**
- 抵押贷最高3000万，业内领先
- 利率最低3.51%，接近国有行水平
- 适合有房产的企业主

**3. 审批相对灵活**
- 对征信和经营数据要求相对灵活
- 小额信用贷审批较快
- 支持多种经营数据（流水/开票/纳税/保单等）

**4. 全国覆盖、网点多**
- 平安银行全国布局
- 线上+线下双渠道
- 服务相对便捷

**5. 科技金融、智能审批**
- 采用AI+大数据评分
- 小额纯信用贷可线上秒批
- 审批效率高

### 产品劣势

**1. 利率分化大**
- 纯信用贷利率4%-8%，跨度大
- 条件一般的客户可能拿到较高利率
- 需要仔细比较其他银行

**2. 额度算法不透明**
- 综合评分模型，难以精准预估
- 同样条件的企业，额度可能差异较大
- 需要实际申请后才知道

**3. 纯信用额度相对保守**
- 纯信用最高200万，低于部分银行（如兴业500万）
- 小微企业实际获批额度可能更低
- 需要大额资金的企业可能不够

**4. 部分产品需要平安关系**
- 保单贷需要有平安保单
- 存量客户更容易审批通过
- 新客户可能额度较保守

### 适用场景

**最适合**：
1. 有房产可抵押，需要大额低息贷款的企业（抵押贷）
2. 有平安保单或公积金，需要小额周转的企业
3. 条件一般，在其他银行难以获批的企业（审批相对灵活）
4. 需要快速资金周转的企业（小额信用贷审批快）
5. 平安银行存量客户

**不适合**：
1. 条件优秀，希望获得最低利率的企业（建议优先国有行）
2. 需要200万以上纯信用额度的企业
3. 成立<1年的初创企业

### 注意事项

⚠️ **关于产品选择**
- 平安有多种小微企业贷款产品
- 申请时明确告知客户经理企业情况
- 让银行推荐最合适的产品方案

⚠️ **关于利率**
- 纯信用贷利率差异大（4%-8%）
- 申请时可以多问几家银行比较
- 抵押贷利率相对优惠且稳定

⚠️ **关于额度预估**
- 系统给的预估额度仅供参考
- 最终额度以审批为准
- 可能低于预估，也可能高于预估

⚠️ **关于抵押贷**
- 如有房产，强烈建议申请抵押贷
- 抵押贷额度高、利率低、期限长
- 值得支付评估费和办理抵押登记

⚠️ **关于平安关系**
- 有平安保单、存款、贷款记录的客户更容易审批
- 可以考虑先开立平安账户，积累关系后申请

## 八、常见问题

**Q1：平安银行小微企业贷款有哪些产品？**

A：主要有纯信用贷、抵押贷、保单贷、公积金贷等。具体申请哪个产品，需要根据企业情况由平安客户经理评估推荐。

**Q2：纯信用贷和抵押贷利率差多少？**

A：纯信用贷4%-8%，抵押贷3.51%-6%。抵押贷利率明显更低，如有房产建议优先考虑抵押贷。

**Q3：可以提前还款吗？**

A：可以，支持提前还款且无违约金。利息按实际使用天数或期数计算。

**Q4：抵押贷需要多久？**

A：一般7-15天，包含房产评估和抵押登记时间。如果房产在外地，时间可能更长。

**Q5：额度可以循环使用吗？**

A：部分产品支持循环使用（如平安普惠额度），具体需咨询客户经理。

**Q6：可以同时申请多种产品吗？**

A：理论上可以，但银行通常会推荐最适合的一种。多产品同时申请可能影响征信查询次数。

**Q7：个体工商户可以申请吗？**

A：可以。个体工商户和小微企业的申请条件基本一致。

**Q8：审批未通过可以再次申请吗？**

A：可以，但建议间隔3-6个月，期间改善经营数据和征信后再申请。
