# 外部平台产品池

**创建日期**：2025-12-07  
**用途**：收集外部平台上的银行产品，作为产品总表的补充来源

---

## 📋 目录说明

本目录用于存放从外部平台收集的银行产品总表。

**定位**：
- 外部平台 = **产品发现池**（不是正式产品库）
- 01_银行政策库 = **我们的标准产品库**
- 通过`product_id`对应，避免混乱

---

## 📁 文件命名规范

格式：`[平台名称]-产品总表.md`

示例：
- `重庆北碚-产品总表.md`
- `沈阳信易贷-产品总表.md`
- `公共采购信易贷-产品总表.md`

---

## 📝 文档模板

每个平台产品总表应包含：

```markdown
# [平台名称] - 产品总表

**来源网址**：xxx  
**整理日期**：2025-xx-xx  
**适用地区**：xx市/xx省  
**合作金融机构数**：xx家

## 产品清单

| 外部产品名 | 所属银行 | 大类 | 平台备注 | 是否已纳入01_银行政策库 | 对应内部product_id |
|-----------|---------|------|---------|----------------------|-------------------|
| ... | ... | ... | ... | 待定 | - |

## 值得关注的产品

**产品1：[产品名称]**
- 特点：[简要说明]
- 是否建议纳入正式产品库：是/否
- 理由：[...]

## 整理说明

- 本平台共xx家合作银行，覆盖税贷/发票贷/抵押贷等xx类产品
- 已挑选xx个产品纳入正式银行政策库
- 其余产品作为参考，暂不收录
```
