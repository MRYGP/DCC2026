## 二、平安银行-速票贷

### 2.1 基本信息（网络可查 ✅）

| 字段 | 内容 |
|------|------|
| 产品名称 | 速票贷 |
| 所属银行 | 平安银行 |
| 产品定位 | 面向开具增值税发票的中小企业主 |
| 授信依据 | 企业开票行为 |
| 贷款性质 | 自然人互联网信用贷款 |

### 2.2 产品特点

- **免**：纯信用、免担保、免抵押
- **高**：最高额度50万（开通数保贷地区，增信后最高可达500万）
- **快**：最快当天放款
- **省**：无需提供纸质资料

### 2.3 申请资格（官方条件）

1. **申请人要求**：
   - 企业法定代表人
   - 必须占股 或 前两大自然人股东之一
   - 年龄不超过65周岁

2. **企业要求**：
   - 至少有**18个月**的增值税开票记录
   - 与交易对手合作稳定

3. **征信要求**：
   - 企业及申请人无不良信用记录
   - 无欠缴记录

## 三、招商银行-招企贷

### 3.1 基本信息（网络可查 ✅）

| 字段 | 内容 |
|------|------|
| 产品名称 | 招企贷 |
| 所属银行 | 招商银行 |
| 产品定位 | 中小企业数据融资 |
| 授信依据 | 公共数据 + 行内交易数据 |
| 贷款性质 | 流动资金贷款 |

### 3.2 产品特点

- **轻**：最高额度300万，无需抵押或担保公司
- **便**：全流程线上操作，无人工审批
- **快**：首笔贷款从申请到放款最快半天
- **活**：随借随还，每日计息

### 3.3 申请条件（官方条件）

1. **担保方式**：
   - 仅需企业法定代表人个人担保
   - 无需抵质押
   - 无需担保公司

2. **操作方式**：
   - 登录"招行企业银行App"线上申请
   - 风控模型自动核额
   - 全流程无人工干预

3. **额度与期限**：
   - 授信额度有效期：一年
   - 单笔最长融资期限：6个月
   - 可循环使用

4. **还款方式**：
   - 每月付息，到期一次还本

### 3.4 利率信息

- 利率：**4%起**（风险定价）
- 无手续费

### 3.5 限制条件

- 企业法人未在招行办理个人经营贷（不影响额度审批，但提款前须结清）

## 六、调研结论

### 6.1 产品推荐优先级

| 优先级 | 银行产品 | 推荐理由 |
|-------|---------|---------|
| ⭐⭐⭐⭐⭐ | 平安银行-速票贷 | 明确以开票为依据，纯信用，最快当天放款 |
| ⭐⭐⭐⭐ | 招商银行-招企贷 | 额度高（300万），全线上，利率4%起 |
| ⭐⭐⭐ | 交通银行-惠商贷 | 待进一步了解具体条件 |

### 6.2 H业务匹配度分析

| 银行产品 | H业务匹配度 | 原因 |
|---------|-----------|------|
| 平安速票贷 | ⭐⭐⭐⭐⭐ | 明确要求18个月开票记录，H业务正好可以做到 |
| 招商招企贷 | ⭐⭐⭐⭐ | 依据交易数据，H业务产生的交易记录有帮助 |
| 交行惠商贷 | ⭐⭐⭐ | 需要验证是否看开票数据 |

### 6.3 关键发现

1. **平安速票贷要求18个月开票记录**
   - 这意味着H业务需要做**至少18个月**才能申请平安速票贷
   - 或者客户已有部分开票记录，H业务补足剩余部分

2. **招商招企贷可能更快**
   - 依据"公共数据+行内交易数据"
   - 可能不需要18个月开票记录
   - 需要验证具体要求

3. **额度差异大**
   - 平安：50万（增信后500万）
   - 招商：300万
   - 交行：待验证
