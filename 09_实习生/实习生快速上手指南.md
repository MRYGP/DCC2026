## 💡 使用Claude Code的技巧

### 技巧1：描述需求要清晰

**❌ 不好的描述**：
```
做一个导入功能
```

**✅ 好的描述**：
```
实现批量导入Excel功能：

功能：
1. 接收上传的xlsx文件
2. 验证13个必填字段（参考V0开发计划）
3. 字段缺失时返回详细错误
4. 批量插入数据库
5. 返回成功/失败统计

技术：
- 使用pandas读取Excel
- 使用Pydantic验证数据
- 使用SQLAlchemy批量插入
- API路径：POST /api/v1/customers/upload

响应格式要符合项目规范（参考.claude/CLAUDE.md）
```

### 技巧2：遇到问题先让AI分析

**不要直接改代码**，先问：
```
这段代码有个问题：[粘贴代码]

错误信息：[粘贴完整错误]

帮我分析问题在哪里，应该怎么修复
```

### 技巧3：使用检查点保护

**开始新功能前**：
- Claude Code自动创建检查点

**出问题时**：
- 按两次`Esc`回退
- 或输入`/rewind`

### 技巧4：让AI解释代码

**看不懂代码时**：
```
解释一下这段代码的逻辑：
[粘贴代码]

用简单的话说明：
1. 这段代码做什么
2. 关键步骤是什么
3. 有什么要注意的
```

### 技巧5：并行开发多个功能

**独立功能可以并行**：
```
使用子代理并行实现这3个功能：

子代理1：批量导入Excel
子代理2：自动诊断分层
子代理3：三段式输出

每个功能独立实现，最后汇总给我
```
