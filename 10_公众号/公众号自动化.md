## 0. V0阶段对齐说明（必读）

### 当前战场
- **V0目标**：2周交付14项P0功能 + 3个中介试点验证价值
- **北极星指标**：改造方案被实际采用的数量
- **自动化定位**：服务V0内容产出，不是完整的内容自动化系统

### 自动化的P0目标（与V0对齐）
1. **辅助试点招募**：产出试点招募文章（优先级最高）
2. **方法论标准化**：把"六宫格+三段式+任务化"模板化
3. **提升产出效率**：从8小时/篇 → 2-3小时/篇（含审核）

### V0阶段约束
- ⚠️ **受众限制**：只服务贷款中介（1-5人小团队），不面向普通借款人
  - **V0不服务**：企业主（让中介服务）、大型中介（他们有系统）、财务顾问/信贷经理
- ⚠️ **合规红线**：不做诱导分享、不排名银行产品、不承诺额度利率
- ⚠️ **数据真实性**：无真实数据前，不发布"激活率21.3%"等不可验证数字
- ⚠️ **资源约束**：1人兼职，预算2000-3000元/月，7天内可用，每周2篇可持续
- ⚠️ **时间紧迫**：2周交付V0，自动化系统必须7天内可用

### V0阶段策略
- **最小实现**：只做MVP，能辅助产出2篇/周即可
- **延后功能**：SEO优化、多引擎搜索、智能配图等延后到V1
- **人工为主**：自动化只是辅助，人工审核占比≥50%

```
Stage 1：数据准备（脚本执行）
```

#### 3. 分发优化系统设计（公众号场景）

**标题AB测试生成**：
```python
class TitleABGenerator:
    def generate_title_options(self, topic):
        """生成多个标题选项用于AB测试"""
        titles = []
        # 痛点共鸣型（不涉承诺）
        titles.append(f"为什么你的{topic}总是达不到银行要求？")
        # 过程导向型
        titles.append(f"解决{topic}问题的3个可执行步骤")
        # 价值承诺型
        titles.append(f"{topic}改善方案：具体任务清单与证据要求")
        # 数字清单型
        titles.append(f"提升{topic}效率的5个关键动作")
        return titles
    
    def compliance_check(self, title):
        """检查标题合规性"""
        forbidden_phrases = ["保证", "包过", "必下款", "100%通过", "秒批"]
        for phrase in forbidden_phrases:
            if phrase in title:
                return False, f"包含禁止表述: {phrase}"
        return True, "合规"
```

### 合规检查工具（自动化）
```python
class ComplianceChecker:
    def __init__(self):
        self.forbidden_phrases = [
            "保证通过", "包下款", "100%通过", "秒批",
            "最低利率", "最快审批", "必过", "一定能",
            "哪家最低", "哪家最快", "XX银行最好"
        ]
        
    def check_article(self, content):
        """检查文章合规性"""
        issues = []
        
        # 检查1：承诺性表述
        for phrase in self.forbidden_phrases:
            if phrase in content:
                issues.append(f"包含禁止表述: {phrase}")
        
        # 检查2：免责声明
        if "免责声明" not in content:
            issues.append("缺少免责声明")
        
        # 检查3：数据来源
        if "平均" in content or "百分比" in content:
            if "信息来源" not in content:
                issues.append("缺少数据来源标注")
        
        # 检查4：诱导分享
        forbidden_share = ["转发海报", "分享得福利", "打卡生成"]
        for phrase in forbidden_share:
            if phrase in content:
                issues.append(f"包含诱导分享: {phrase}")
        
        if issues:
            return False, issues
        return True, "合规检查通过"
```
