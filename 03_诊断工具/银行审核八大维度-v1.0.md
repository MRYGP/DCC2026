## 📌 一句话总结

> 银行在看：
> **"这公司是不是真的？"**（实体经营、社保、资质荣誉）  
> **"这公司现在扛不扛得住？"**（负债、流水、利润留存）  
> **"这个人靠不靠谱？"**（法人风险）  
> **"这生意讲得通吗？"**（生意故事）

## 🎯 八大维度模型

| 维度 | 核心关注点 | 银行视角 | 评分标准（★） |
|------|-----------|---------|-------------|
| **1. 实体经营** | 有真实业务、有发票 | 是不是空壳公司？有没有真实经营？ | ★★★★☆ 强 / ★★★☆☆ 中 / ★★☆☆☆ 弱 |
| **2. 负债结构** | 法人+企业负债情况 | 现在扛不扛得住？会不会还不起？ | ★★★★☆ 健康 / ★★★☆☆ 一般 / ★★☆☆☆ 偏重 |
| **3. 流水体质** | 规模+质量 | 流水能不能支撑申贷额度？ | ★★★★☆ 充足 / ★★★☆☆ 一般 / ★★☆☆☆ 不足 |
| **4. 社保信号** | 至少3个员工 | 是不是"真有团队"？ | ★★★★☆ 3人以上 / ★★☆☆☆ 1-2人 / ★☆☆☆☆ 无 |
| **5. 资质荣誉** | 科小/高新/专精特新等 | 有没有"硬背书"？ | ★★★★☆ 多项 / ★★★☆☆ 1-2项 / ★★☆☆☆ 无 |
| **6. 利润留存** | 利润+资金留存+结息 | 账上有没有钱？是不是当天进当天出？ | ★★★★☆ 有留存+结息 / ★★★☆☆ 偶有留存 / ★★☆☆☆ 基本无 |
| **7. 法人风险** | 个人风险画像 | 这个人靠不靠谱？ | ★★★★☆ 无不良 / ★★★☆☆ 轻微 / ★★☆☆☆ 有风险 |
| **8. 生意故事** | 赚钱逻辑 | 这生意讲得通吗？ | ★★★★☆ 清晰 / ★★★☆☆ 一般 / ★★☆☆☆ 模糊 |

## 📋 对应体检问卷问题

### 1. 实体经营 & 发票

**问题1**：公司目前主要经营场景（多选）
- [ ] 有固定办公场地
- [ ] 有厂房/门店
- [ ] 主要线上办公
- [ ] 其他：__________

**问题2**：是否持续有开具增值税发票？
- [ ] 是，最近12个月每月都有
- [ ] 有，但不太规律
- [ ] 基本不开票

**问题3**：发票主要类型
- [ ] 专票为主
- [ ] 普票为主
- [ ] 两者都有

**评分逻辑**：
- 有固定场地 + 每月开票 + 专票为主 → ★★★★☆
- 有场地 + 不规律开票 → ★★★☆☆
- 线上办公 + 基本不开票 → ★★☆☆☆

---

### 2. 负债（法人 + 企业）

**法人负债：**

**问题4**：你个人名下每月总还款（房贷+车贷+消费贷+信用卡）约：______ 元

**问题5**：是否有正在使用的小贷/网贷？
- [ ] 有，大约 ____ 万
- [ ] 无

**企业负债：**

**问题6**：公司目前在几家银行/机构有贷款或授信？
- [ ] 0家
- [ ] 1-2家
- [ ] 3-4家
- [ ] 5家以上

**问题7**：合计授信额度约：_____ 万，已用约：_____ 万

**评分逻辑**：
- 法人无小贷 + 企业1-2家授信 + 使用率<50% → ★★★★☆
- 法人有小贷但不多 + 企业3-4家授信 → ★★★☆☆
- 法人小贷多 + 企业5家以上授信 + 使用率高 → ★★☆☆☆

---

### 3. 流水

**问题8**：公司近12个月对公账户月均进账约：
- [ ] < 50万
- [ ] 50-200万
- [ ] 200-500万
- [ ] 500万以上

**问题9**：前3大客户回款占总回款比例约：
- [ ] <30%
- [ ] 30-70%
- [ ] >70%

**问题10**：流水是否经常出现"当天进当天出"的情况？
- [ ] 很少，经常有余额过夜
- [ ] 偶尔
- [ ] 经常，基本不留余额

**评分逻辑**：
- 月均200万以上 + 客户分散（<30%）+ 有余额留存 → ★★★★☆
- 月均50-200万 + 客户集中度中等 → ★★★☆☆
- 月均<50万 + 客户集中度高（>70%）+ 当天进当天出 → ★★☆☆☆

---

### 4. 社保（至少3人）

**问题11**：当前为多少名员工缴纳社保？
- [ ] 无
- [ ] 1-2人
- [ ] 3-5人
- [ ] 5人以上

**问题12**：缴纳社保的持续时间：
- [ ] 不满6个月
- [ ] 6-12个月
- [ ] 12个月以上

**评分逻辑**：
- 3人以上 + 持续12个月以上 → ★★★★☆
- 3-5人 + 持续6-12个月 → ★★★☆☆
- 1-2人或无 → ★★☆☆☆

**银行视角**：
- 0人 → 明显减分
- 3人以上且持续1年 → 明显加分

---

### 5. 资质 / 荣誉

**问题13**：公司是否拥有以下资质或荣誉（多选）：
- [ ] 科技型中小企业（科小）
- [ ] 高新技术企业
- [ ] 小巨人/专精特新/龙头企业
- [ ] A级纳税人
- [ ] 其他政府荣誉：________________
- [ ] 以上都没有

**评分逻辑**：
- 高新+专精特新+A级纳税人 → ★★★★☆
- 科小或A级纳税人（1-2项） → ★★★☆☆
- 无任何资质 → ★★☆☆☆

**说明**：没有也没关系，但有的话要帮他"高亮"。

---

### 6. 利润 & 资金留存（结息）

**问题14**：公司过去两年利润大致情况：
- [ ] 持续盈利
- [ ] 有一年微利/收支平衡
- [ ] 经常亏损

**问题15**：公司对公账户是否经常留有较大余额过夜？
- [ ] 经常有（且有结息记录）
- [ ] 偶尔
- [ ] 基本没有，进出很快

**评分逻辑**：
- 持续盈利 + 经常有余额+结息 → ★★★★☆
- 微利/平衡 + 偶尔有余额 → ★★★☆☆
- 经常亏损 + 基本无余额 → ★★☆☆☆

---

### 7. 法人风险画像（人品）

**问题16**：法人本人是否存在以下情况（多选）：
- [ ] 有严重征信逾期（超过90天）
- [ ] 曾被法院列为失信被执行人
- [ ] 曾涉及刑事案件/被判刑
- [ ] 有长期赌博、巨额网贷等情况
- [ ] 以上都没有

**评分逻辑**：
- 以上都没有 → ★★★★☆
- 有轻微逾期但已还清 → ★★★☆☆
- 有严重逾期/失信/刑事案件 → ★★☆☆☆

**说明**：这块数据最好未来由系统查询，但问一嘴可以先做自我申报。

### 8. 故事（为什么赚钱）

**问题17**：请用3-5句话，简单说明：
1. 你们是做什么的？
2. 主要客户是谁？
3. 为什么这个生意能赚钱？
4. 未来12个月这笔贷款主要用在哪里？

**回答**：_________________________________________________________

**评分逻辑**：
- 逻辑清晰 + 客户明确 + 用途具体 → ★★★★☆
- 能说清楚但不够具体 → ★★★☆☆
- 说不清楚/逻辑混乱 → ★★☆☆☆

**用途**：
- 一份用于贷查查体检报告里的"老板自己讲的故事"
- 一份作为前端合伙人/你自己，对银行经理讲解时的"底稿"

## 📊 体检报告输出模板

### 银行视角体质小结（示例）

```markdown
【银行视角体质小结】

1. 实体经营：★★★☆（有真实场地+稳定开票）
2. 负债结构：★★（法人小贷稍多，企业多头授信4家）
3. 流水体质：★★★（月均300万，客户集中度中等）
4. 社保信号：★（暂无或仅1-2人，建议逐步补齐至3人以上）
5. 资质荣誉：★★（有科小，无高新）
6. 利润留存：★★☆（账上偶有留存，结息记录较少）
7. 法人风险：★★★★（无重大不良记录）
8. 生意故事：★★★（盈利逻辑清晰，但缺少书面化材料）

综合：当前属于"中等体质"，有贷款空间，但需在【社保】【资金留存】【票税/资质】三块做优化。
```

### 综合评语模板

**体质等级判断**：

| 综合评分 | 体质等级 | 评语模板 |
|---------|---------|---------|
| 32-40星 | **A级（优质）** | 各项指标良好，可直接匹配银行产品，快速放款 |
| 24-31星 | **B级（一般）** | 部分条件不足，但可改造，推荐改造方案后申请 |
| 16-23星 | **C级（较差）** | 多个条件不达标，建议先修复问题，暂缓申请 |
| <16星 | **D级（不适合）** | 当前不适合申请，建议先解决基础问题 |

## 🔗 与现有体系的关系

### 与"贷款体质六宫格"的对应关系

| 八大维度 | 对应六宫格维度 |
|---------|---------------|
| 实体经营 | 行业体质 |
| 负债结构 | 征信体质 |
| 流水体质 | 流水体质 |
| 社保信号 | 票税体质（间接） |
| 资质荣誉 | 资质体质 |
| 利润留存 | 流水体质（间接） |
| 法人风险 | 征信体质 |
| 生意故事 | 综合评估 |

### 与"银行放款五问"的对应关系

| 银行五问 | 对应八大维度 |
|---------|-------------|
| 做什么？ | 实体经营、生意故事 |
| 用途？ | 生意故事 |
| 产值？ | 实体经营、流水体质 |
| 负债？ | 负债结构 |
| 利润？ | 利润留存、流水体质 |

### 使用场景

1. **诊断问卷设计**：把这8个维度的问题直接嵌入问卷
2. **体检报告生成**：输出"银行视角体质小结"
3. **方案推荐**：根据薄弱维度推荐对应改造方案
4. **合伙人培训**：让合伙人理解"银行是怎么看企业的"
