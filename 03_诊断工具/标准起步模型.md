## 一、基本参数

| 参数 | 数值 | 备注 |
|------|------|------|
| 法人资历 | 满6个月 | 部分银行要求满1年，建行/农行6个月可申请 |
| 开票流水 | 200万/月 | 对公账户进账，年化2400万 |
| 不开票流水 | 100-200万/月 | 有流水无票，需来源说明（合同/付款记录） |
| 账户总流水 | 300-400万/月 | 银行实际看到的数字 |
| 利润率 | 0.1% | 贸易中间商差价极薄，行业普遍现象 |
| 增值税率 | 13% | 进项几乎全额抵扣，实际税负率≈0.13% |
| 社保人数 | 2人 × 1600元/月 | 证明实际经营状态 |
| 初始存款 | 20万 | 用于日常税费/社保周转，长期维持账户结息 |
| 银行账户 | 2个 | 基本户 + 一般户 |
| 科小认定 | 办理成本约2万 | 认定后开放科创类信贷产品通道 |
| 流水服务商 | 1000元/月 | 贷款期间持续维护，有起步价 |

## 五、融资演进路径（A→B→续贷）

### 时间轴

```
Month 0 ──── Month 6 ──── Month 12 ──────────── Month 36 ──→ 续贷
  准备期        A阶段下款    升级B阶段              续贷
  6个月         300万        +200万=500万           500万（无中介费）
```

### A阶段（第6个月下款，300万）

**产品路线**：建行信用快贷 / 农行网捷贷（单行最高300万，年化3%起）

| 参数 | 数值 |
|------|------|
| 下款额度 | 300万 |
| 银行年化利率 | 3-3.5% |
| 中介费 | 300万 × 15% = **45万** |
| 贷款期限 | 3年 |

**A阶段6个月成本（下款前准备期）**

| 科目 | 金额 |
|------|------|
| 科小认定 | 20,000元 |
| 流水服务费 | 6,000元 |
| 社保 | 19,200元 |
| 增值税 | 15,600元 |
| **准备期小计** | **60,800元** |

**A阶段贷款期成本（第6-12个月，6个月）**

| 科目 | 计算 | 金额 |
|------|------|------|
| 中介费（一次性） | 300万 × 15% | 450,000元 |
| 银行利息 | 300万 × 3% ÷ 12 × 6 | 45,000元 |
| 流水服务费 | 1000 × 6 | 6,000元 |
| 社保 | 3200 × 6 | 19,200元 |
| 增值税 | 2600 × 6 | 15,600元 |
| **贷款期小计** | | **535,800元** |

### B阶段（第12个月升级，追加200万=500万）

**产品路线**：原行追加 + 第二家银行补充，或科创类产品通道

| 参数 | 数值 |
|------|------|
| 追加额度 | 200万 |
| 中介费 | 200万 × 5% = **10万** |
| 贷款期限 | 3年 |

**B阶段贷款期成本（第12-36个月，24个月）**

| 科目 | 计算 | 金额 |
|------|------|------|
| 中介费（追加200万） | 200万 × 5% | 100,000元 |
| 银行利息 | 500万 × 3% ÷ 12 × 24 | 300,000元 |
| 流水服务费 | 1000 × 24 | 24,000元 |
| 社保 | 3200 × 24 | 76,800元 |
| 增值税 | 2600 × 24 | 62,400元 |
| **B阶段小计** | | **563,200元** |

### 第一轮全周期汇总（Month 0-36）

| 阶段 | 金额 |
|------|------|
| 准备期 | 60,800元 |
| A阶段贷款期 | 535,800元 |
| B阶段（含升级中介费） | 563,200元 |
| **全周期合计** | **约116万元** |
| **中介费合计** | **55万（占比47%）** |
| **银行利息合计** | **34.5万** |

### 续贷阶段（Month 36起，每轮3年）

中介费归零，持续成本大幅下降。

| 科目 | 计算 | 金额 |
|------|------|------|
| 中介费 | **0**（不再收取） | 0元 |
| 银行利息 | 500万 × 3% × 3年 | 450,000元 |
| 流水服务费 | 1000 × 36 | 36,000元 |
| 社保 | 3200 × 36 | 115,200元 |
| 增值税 | 2600 × 36 | 93,600元 |
| **每轮小计** | | **约69.5万元** |

**续贷阶段年化资金成本：69.5万 ÷ 500万 ÷ 3年 ≈ 4.6%**

|------|------|----------|------|----------|

### B阶段补充路线（追加200万，第12个月）

| 路线 | 产品 | 说明 |
|------|------|------|
| 本地城商行叠加 | 长沙银行信用贷 | 本地银行关系更易维护，续贷稳定性强 |
| 第二家全国行 | 农行/工行信用贷 | 多行组合，征信查询需提前规划 |
| 科创通道 | 长沙农商行科技知识价值贷/建行科创贷 | 科小资质加成，利率3%起 |

### 银税贷路线（不推荐）

税负率0.13%，银税贷（长沙银行快乐e贷、农行纳税e贷等）要求近两年平均纳税≥1-5万元且纳税信用等级B级以上。贸易公司税负极低，走此路线审批概率低。**主力走流水贷路线**。

### 湖南/长沙特殊政策（2025年）

- 截至2025年4月末，湖南省银行机构通过小微企业融资协调机制新发放贷款平均利率为4%，较建立机制以来下降1.53个百分点。长沙贷款利率整体低于全国均值，3%可期。
- 湖南省有"潇湘财银贷"政银合作平台，可通过纳税、经营流水等信用数据"以信换贷"，替代传统抵押担保，**贸易公司可尝试申请**。
- 长沙高新区/经开区科技型企业可享园区贴息政策（长沙银行信用贷注明）。科小认定后，如注册在高新区，利率可进一步降低。

| 风险点 | 银行疑虑 | 标准应对口径 |
|--------|----------|--------------|
| 税负率0.13% | 空转嫌疑 | 贸易中间商赚差价极薄，进项几乎全额抵扣，行业普遍现象；主走流水贷，不走银税贷 |

| 多行叠加征信查询 | 征信花 | 申请顺序需规划：先大行后股份行，间隔≥1个月，近6个月查询次数控制在6次以内 |

## 九、诊断结论（ABC分层）

```
体质评级：B级（可贷，有条件）

✅ 通过：流水规模、账户结息、社保、法人资历、科小资质
⚠️ 需补充：不开票流水来源说明材料（合同/付款记录）
⚠️ 需说明：税负率低的行业背景（走流水贷路线规避）

可贷额度预估：
  无科小 → 单行100-160万
  有科小 → 单行+科创 300万（A阶段目标）
  B阶段多行叠加 → 500万（第12个月目标）

目标路径：
  Month 6：科小 + 流水类信用贷（建行/农行）→ 300万 ✅
  Month 12：追加第二家银行或科创通道 → +200万=500万 ✅
  Month 36+：续贷，无中介费，年化成本降至4.6% ✅
```

## 十二、科小认定说明

**定义**：科技型中小企业，由科技部门每年评定一次。

**核心条件**：职工总数<500人 / 年销售额<2亿元 / 资产总额<2亿元 / 研发费用或科技人员占比达标。

**对贷款的作用**：开放建行科创贷、科创助力贷等专项通道；部分产品利率优惠；政府贴息政策适用（上海等地专精特新可申请利息30%贴息）。

**费用与周期**：约2万元（含代办），周期1-3个月；有效期1年，续期约0.5-1万/年。

**ROI测算**：

```
投入：2万元
开放额度：+100-200万（科创通道）
利率节约（优惠0.5% × 300万 × 1年）：约1.5万/年
综合ROI：显著正收益，第一年即回本
```
