**版本**：v1.0  
**创建日期**：2025-12-06  
**适用场景**：贷查查MVP  
**问题数量**：15道题，分4个模块  
**预计填写时间**：3-5分钟

---

## 一、问卷整体设计

### 1.1 设计原则

**简洁性**：
- 每个问题只问一件事
- 选项不超过7个
- 避免专业术语

**渐进性**：
- 从简单到复杂
- 从基础信息到专业数据
- 支持跳过非必填项

**智能性**：
- 根据答案动态跳转
- 自动过滤无关问题
- 智能提示帮助文案

---

### 1.2 问卷流程

```
[欢迎页] 
  ↓
[模块1：基本信息] 4题 → 进度 1/4
  ↓
[模块2：财务信息] 5题 → 进度 2/4
  ↓
[模块3：资产信息] 3题 → 进度 3/4
  ↓
[模块4：特殊条件] 3题 → 进度 4/4
  ↓
[提交页]
```

---

### 1.3 分步展示策略

**每页显示题数**：3-4题  
**进度条**：显示整体进度（如：第2步，共4步）  
**按钮**：
- "下一步"（主按钮）
- "上一步"（次按钮，第一页隐藏）
- "跳过"（可选题显示）

#### Q1：企业名称（可选）

```

```

**输入类型**：文本框  
**字段名**：`company_name`  
**数据类型**：String  
**是否必填**：❌ 否  
**验证规则**：
- 最大长度：50字符
- 允许为空

**前端交互**：
- 占位符文本："例如：湖南三湘科技有限公司"
- 获得焦点时清空占位符
- 失去焦点时保存输入

**数据示例**：
```json
{
  "company_name": "湖南三湘科技有限公司"
}
```

**问题文案**：

### 💰 模块2：财务信息（5题）

**目标**：评估企业财务状况，匹配发票贷、税贷等产品

> 银行在做发票贷/税贷授信时，会同时参考：
> - 上一纳税年度（完整年度）的开票额/纳税额（决定基础额度）
> - 最近12-24个月的开票和纳税情况（决定是否放款、是否缩减额度）
>
> 因此问卷分别询问【去年全年】和【今年截至目前】的数据，用于做趋势判断。

#### Q6：年开票金额（去年 & 今年截至目前）

**问题文案**：
```
Q6-1：企业【去年全年】开具的增值税发票总金额大约是多少？
Q6-2：企业【今年截至目前】（含本月）的增值税发票总金额大约是多少？
```

**帮助文案**：
```
💡 银行做发票贷/税贷时，会同时参考上一年度和最近12个月的开票数据
💡 这里给出大致区间即可，不需要精确到元
💡 开票金额 = 开给客户的所有增值税发票合计，可登录电子税务局查询
💡 如果没有开过票，请选择"无开票"
```

**输入类型**：下拉选择  
**字段名**：
- `annual_invoice_amount_last_year`
- `invoice_amount_ytd`
**数据类型**：String  
**是否必填**：✅ 是  

**选项设计**（两道题共用）：
```
无开票
100万以下
100-300万
300-500万
500-1000万
1000-3000万
3000万以上
```

**枚举值定义**（复用一套枚举）：
```json
{
  "no_invoice": "无开票",
  "less_than_100w": "100万以下",
  "100w_to_300w": "100-300万",
  "300w_to_500w": "300-500万",
  "500w_to_1000w": "500-1000万",
  "1000w_to_3000w": "1000-3000万",
  "more_than_3000w": "3000万以上"
}
```

**影响说明**：
- 可用额度粗估：`max(annual_invoice_amount_last_year, invoice_amount_ytd)` 作为近12个月的额度锚点
- 趋势判断：`invoice_amount_ytd / annual_invoice_amount_last_year`，若 <50% 视为明显下滑
- `no_invoice` 或 `less_than_100w`：不符合发票贷条件，提示"开票提升术"
- `100w_to_300w`：符合部分发票贷，额度较低
- `300w_to_500w`及以上：符合多数发票贷，额度较高

**数据示例**：
```json
{
  "annual_invoice_amount_last_year": "300w_to_500w",
  "annual_invoice_amount_last_year_label": "300-500万",
  "invoice_amount_ytd": "100w_to_300w",
  "invoice_amount_ytd_label": "100-300万"
}
```

#### Q13：法人特殊身份

```
企业法定代表人是否属于以下特殊群体？
```

```
💡 符合条件的可以申请专项贷款，享受财政贴息
💡 可以多选，如果都不符合请选"都不是"
```

**输入类型**：多选框  
**字段名**：`special_identity`  
**数据类型**：Array  
**是否必填**：✅ 是（至少选一项）  

**选项设计**：
```
☐ 高校毕业生（毕业5年内）
☐ 退伍军人
☐ 残疾人
☐ 都不是
```

**枚举值定义**：
```json
{
  "graduate": "高校毕业生（毕业5年内）",
  "veteran": "退伍军人",
  "disabled": "残疾人",
  "none": "都不是"
}
```

**验证规则**：
- 如果选择"都不是"，不能同时选择其他选项
- 如果选择其他选项，不能选择"都不是"

**影响说明**：
- `graduate`：可申请大学生创业贷，最高30万个人/400万企业，财政贴息
- `veteran`：可申请退伍军人创业贷，最高30万个人/500万企业，财政贴息
- `disabled`：可申请残疾人创业贷，具体政策各地不同
- `none`：无特殊身份优惠

**数据示例**：
