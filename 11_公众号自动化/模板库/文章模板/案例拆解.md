# 【案例拆解】文章模板

## 模板基本信息
- **模板类型**：案例拆解
- **目标读者**：贷款中介、业务员
- **阅读时间**：5-8分钟
- **核心价值**：提供可复制的成功经验
- **模板版本**：v1.0 (2026-02-03)

## 标题公式

### 合规标题格式
```
《[企业类型]+[贷款类型]+[核心问题]：[过程方案]+[指标改善]》
```
**核心原则**：描述过程与指标改善，绝不承诺具体贷款结果

### 合规标题示例
1. **痛点共鸣型**：《XX科技企业税贷额度不足：3个月开票提升50%的完整改造路径》
2. **问题解决型**：《征信花了怎么办？90天"征信修复"任务清单与证据要求》
3. **数字清单型**：《激活存量客户的5个关键步骤：从睡眠到可贷的完整路径》
4. **过程导向型**：《月入3万 vs 月入30万中介的差距：具体任务与时间投入对比》

### 禁语清单（必须规避）
以下表述在标题和正文中都**绝对禁止**：
- ❌ 保证/包过/必下款/秒批/100%通过
- ❌ 最低利率/最高额度/最优方案
- ❌ 具体贷款额度数值（如"贷到200万"）
- ❌ 确定时间承诺（如"90天拿到贷款"）
- ❌ 内部渠道/特殊关系/独家方案

### 合规表述替代
将结果承诺改为过程与边界描述：
- ❌ "90天贷到200万" → ✅ "90天把'票税体质'补到可申条件：任务清单+证据要求"
- ❌ "保证通过" → ✅ "达到银行准入标准的具体步骤"
- ❌ "最低利率" → ✅ "符合当前市场水平的利率区间"

## 文章结构（共6部分，约1500字）

### 第一部分：案例背景（200-300字）
**目的**：建立共鸣，让读者代入

**合规要点**：
- 使用虚拟代号：客户A001、企业B002等
- 数值使用区间：年开票300-500万、注册资本500-1000万
- 避免可识别组合：不出现"深圳南山科技公司年开票500万"这类组合
- 保持真实性但可复算：区间合理，任务可执行

**内容要点**：
```
【企业画像（脱敏版）】
- 企业代号：[客户A001/B002等]
- 成立时间区间：[3-5年/5-8年等]
- 注册资本区间：[300-500万元]
- 主营行业：[科技/贸易/制造等大类]

【原始状态（区间描述）】
- 贷款需求区间：[300-500万元]
- 当前状况：[具体问题描述，如"票税体质不足"、"征信查询过多"]
- 瓶颈分析：[具体差距，如"年纳税距产品要求差20-30万"]
```

**写作提示**：
- 用区间数字说话，避免精确值
- 突出"典型性"（很多中介都遇到过类似问题）
- 设置悬念（这么难的情况，后来怎么样？）
- **绝对不出现**：真实公司名、详细地址、唯一业务名

### 第二部分：问题诊断（300-400字）
**目的**：展示专业性，建立信任

**内容要点**：
```
【问题清单】
1. 硬性问题：[如开票不足、征信问题等]
2. 软性问题：[如材料不全、沟通不畅等]
3. 认知误区：[客户或中介的错误认知]

【量化分析】
- 目标产品要求：[具体准入条件]
- 客户当前差距：[具体数值差距]
- 关键瓶颈点：[影响最大的问题]
```

**写作提示**：
- 用表格对比更清晰
- 引用银行具体准入标准
- 突出"诊断过程"的专业性

### 第三部分：改造方案（400-500字）
**目的**：提供可操作的解决方案

**内容要点**：
```
【方案设计】
1. 核心措施：[最重要的改造动作]
2. 辅助措施：[辅助性优化]
3. 时间规划：[分阶段执行计划]

【执行细节】
- 具体操作步骤：[每一步做什么]
- 资源需求：[需要什么工具/服务商]
- 预期效果：[每一步的预期结果]
```

**写作提示**：
- 分步骤说明，让读者能跟着做
- 提供工具/模板下载链接（如果适用）
- 强调"可执行性"

### 第四部分：执行过程（300-400字）
**目的**：展示实操细节，增加可信度

**内容要点**：
```
【执行挑战】
1. 遇到的困难：[具体问题]
2. 解决方案：[如何克服]
3. 关键决策：[重要的选择点]

【时间线】
- 第1周：[做了什么]
- 第1个月：[进展如何]
- 第3个月：[最终结果]
```

**写作提示**：
- 展示"真实感"（不是一帆风顺）
- 强调"坚持执行"的重要性
- 可以加入客户/中介的真实反馈

### 第五部分：最终结果（200-300字）
**目的**：展示成果，增强说服力

**内容要点**：
```
【数据对比表】
| 指标 | 改造前 | 改造后 | 提升幅度 |
|------|--------|--------|----------|
| 年开票 | [数值] | [数值] | [比例] |
| 征信评分 | [分数] | [分数] | [比例] |
| 可贷额度 | [数值] | [数值] | [比例] |

【实际成果】
- 最终贷款：[银行名称+产品+额度+利率]
- 时间成本：[从开始到成功的时间]
- 经济收益：[中介服务费/客户价值]
```

**写作提示**：
- 数据对比要直观清晰
- 强调"投入产出比"
- 可以展示银行审批截图（脱敏后）

### 第六部分：经验启示（200-300字）
**目的**：提炼经验，提供价值延伸

**内容要点**：
```
【可复制的经验】
1. 核心方法：[最关键的成功因素]
2. 适用场景：[什么情况下可以用这个方法]
3. 注意事项：[使用时的避坑指南]

【给中介的建议】
- 执行建议：[具体操作建议]
- 资源推荐：[推荐工具/服务商]
- 下一步：[成功后的后续动作]
```

**写作提示**：
- 提炼"通用性"经验
- 提供"马上能用"的建议
- 设置互动环节（提问/反馈）

## 互动元素设计

### 文末互动模板
```
---

**互动环节**

1. **提问**：你在客户改造中遇到过什么问题？
2. **福利**：关注后回复"案例模板"，获取《客户改造案例记录表》
3. **社群**：加入"中介增长交流群"，一起讨论更多案例

**免责声明**
⚠️ 本案例为真实经历改编，具体操作请结合实际情况。贷款结果以银行审批为准。
```

### 内文互动设计
- **思考题**：在关键决策点设置"如果是你，会怎么做？"
- **小贴士**：在复杂步骤旁加"小贴士"提醒注意事项
- **模板下载**：在工具介绍处提供下载链接

## SEO优化要点

### 关键词布局
- **主关键词**：贷款中介案例、客户改造、开票提升、征信修复
- **长尾关键词**：XX银行税贷案例、中小企业贷款成功案例
- **位置要求**：标题、开头、小标题、结尾都要出现关键词

### 内链建议
- 关联文章：[相关主题的历史文章]
- 工具推荐：[相关工具的详细介绍]
- 知识库链接：[相关知识库文章]

### 元数据优化
- **摘要**：150字左右，包含核心成果和关键词
- **标签**：贷款案例、中介经验、客户改造、银行产品
- **封面图**：建议使用数据对比图表

## 审核检查清单

### 事实准确性检查
- [ ] 所有数据都有来源依据
- [ ] 案例细节真实可信
- [ ] 时间逻辑合理
- [ ] 银行产品信息准确

### 合规性检查
- [ ] 包含完整免责声明
- [ ] 无承诺性表述（100%通过、保证额度等）
- [ ] 客户信息已脱敏
- [ ] 不涉及敏感话题

### 专业性检查
- [ ] 贷款逻辑严密
- [ ] 建议切实可行
- [ ] 术语使用准确
- [ ] 符合行业规范

### 可读性检查
- [ ] 结构清晰易读
- [ ] 语言通俗易懂
- [ ] 重点突出明确
- [ ] 互动元素恰当

## 模板使用说明

### 如何使用本模板
1. **选择案例**：从知识库中选择一个有代表性的成功案例
2. **收集素材**：整理案例相关数据、截图、客户反馈等
3. **填充模板**：按照6部分结构填充具体内容
4. **优化调整**：根据实际情况调整篇幅和重点
5. **审核发布**：通过审核检查后发布

### 模板定制建议
- **行业细分**：可以针对不同行业（科技、贸易、制造）定制子模板
- **问题类型**：可以针对不同问题（开票、征信、资质）定制子模板
- **中介规模**：可以针对不同规模中介（1人、3-5人、10+人）定制子模板
