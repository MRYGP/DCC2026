# 贷款中介行业术语黑名单 v1.0
# 以下术语在文章中必须解释或替换

## 核心原则
1. **首次出现必须解释**：术语第一次出现时，必须在括号内解释
2. **高频术语可简化**：文章内多次出现，第二次起可省略解释
3. **替换优先**：能用白话说清楚的，不用术语
4. **合规优先**：涉及监管的术语必须解释清楚，避免误解

## 术语列表（必须解释或替换）

### 流水体质相关
对公流水 → 企业银行账户的进出账记录
流水走账 → 银行账户的资金流动
空转嫌疑 → 资金进来立刻转出，没有真实业务
流水集中度 → 主要客户占比（前五大客户的收款占比）
月均流水 → 最近6个月银行账户的平均进账金额
流水稳定性 → 每个月流水金额是否稳定，波动大不大

### 票税体质相关
开票额 → 企业开出的发票总金额
纳税额 → 企业实际缴纳的税款
票税比 → 开票额与纳税额的比例（比如开票100万，纳税10万，票税比10%）
零申报 → 当月收入为0，纳税为0
负申报 → 当月亏损，申请退税
进项票 → 公司买东西收到的发票（可以抵扣税款）
销项票 → 公司卖东西开出的发票（需要缴纳税款）

### 征信体质相关
征信花 → 征信查询次数过多（近6个月>10次）
多头授信 → 在多家银行同时有贷款
小贷占比 → 小额贷款公司贷款占总负债比例
M1/M3逾期 → 逾期1个月/3个月（M=Month）
代偿 → 担保公司代替你还款（严重问题）
负债率 → 总负债除以总资产的比例（比如50%）
信用卡使用率 → 信用卡已用额度占总额度的比例

### 产品相关
受托支付 → 银行直接把钱打给供应商（不是打给你）
LPR → 贷款市场报价利率（基准利率）
抵押率 → 抵押物价值的可贷比例（如70%）
准入条件 → 银行要求客户必须满足的条件
额度 → 银行批准给你的贷款金额
期限 → 贷款可以用多长时间（如1年、3年）
还款方式 → 每月还多少、怎么还（等额本息、先息后本）

### 改造相关
H业务 → 开票业务（帮助企业增加开票额）
资质认定 → 高新技术企业、专精特新等官方认证
过桥 → 临时借钱还银行贷款，然后重新贷出来
垫资 → 暂时借一笔钱给客户，让他满足银行条件
实缴 → 注册资本实际到账，不是只写个数字
专利实缴 → 用专利技术评估作价，代替现金出资

### 银行政策相关
银行白名单 → 银行特别喜欢的行业或客户类型
行业限制 → 某些行业银行不给贷款（如房地产、娱乐）
风险偏好 → 银行喜欢什么样的客户，不喜欢什么样的
授信额度 → 银行愿意给你的最高贷款金额
用信额度 → 你实际使用的贷款金额（可能小于授信）
授信有效期 → 银行批准的额度可以用多长时间（如1年）

### 中介业务相关
返点 → 银行给中介的佣金（按贷款金额比例）
渠道费 → 客户给中介的服务费
前端 → 跟客户接触、收资料的人
后端 → 跟银行沟通、做材料的人
跑单 → 客户跳过中介直接找银行
同行单 → 别的中介做不了转给你的客户

## 禁止直接使用的表述（合规风险高）

### 承诺性表述（绝对禁止）
- "保证通过"
- "包下款"
- "100%通过"
- "必过"
- "一定能"
- "肯定可以"

### 夸大性表述（谨慎使用）
- "最低利率"（必须说明是哪家银行什么产品）
- "最快审批"（必须说明时间前提和条件）
- "额度最高"（必须说明是和什么对比）
- "门槛最低"（必须说明具体门槛）

### 虚构性表述（禁止使用）
- "真实故事"（除非有客户授权和匿名化处理）
- "亲身经历"（除非真实发生且有证据）
- "我有个朋友"（除非真的有这样的朋友）

### 违规操作表述（禁止使用）
- "虚开发票"
- "做假账"
- "改流水"
- "包装客户"
- "买流水"
- "做假材料"

## 推荐替换表述

### 可用表述
- "通过率较高"（基于历史数据）
- "条件相对宽松"（基于银行政策）
- "审批速度较快"（基于同业对比）
- "额度相对充足"（基于产品特点）

### 中性表述
- "满足条件就可能通过"
- "材料齐全审批会更快"
- "额度取决于综合评估"
- "利率根据征信和资质决定"

### 风险提示表述
- "具体以银行审批为准"
- "需要银行最终审核"
- "根据客户具体条件决定"
- "以银行实际批复为准"

## 使用指南

### 检查流程
1. 写作时：避免使用黑名单中的术语
2. 编辑时：检查是否解释了必要的术语
3. 发布前：用可读性检查工具验证
4. 复盘时：分析术语使用对完读率的影响

### 阈值设置
- 每篇文章：术语命中≤5个
- 每个段落：术语≤2个
- 整个文档：平均句长≤35字

### 例外情况
以下情况可适当放宽：
- 深度专业文章（目标用户是资深中介）
- 行业研究报告（需要专业术语）
- 内部培训材料（受众熟悉术语）
