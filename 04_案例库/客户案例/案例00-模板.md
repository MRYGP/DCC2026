# 📋 案例模板

**版本**：v1.0  
**创建日期**：2025-12-06  
**用途**：标准化案例记录格式  
**适用对象**：业务团队、产品团队

---

## 📌 使用说明

### 模板目的

1. **统一格式**：确保所有案例记录格式一致
2. **信息完整**：确保关键信息不遗漏
3. **快速复用**：方便生成营销材料、培训资料
4. **数据积累**：为算法优化提供真实数据

### 填写原则

✅ **真实性**：所有数据必须真实可靠  
✅ **完整性**：必填字段必须填写  
✅ **合规性**：遵守合规要求，避免承诺性表述  
✅ **可读性**：描述清晰，数据准确

### 案例分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 成功案例 | 改造成功，贷款增额 | 制造企业发票贷增额 |
| 失败案例 | 改造未成功或效果不佳 | 房地产企业被拒 |
| 进行中案例 | 正在改造，尚未完成 | 科技企业高新认定中 |

### 4.2 投入产出分析

**总投入**：
- H业务服务费：XX万
- 税费成本：XX万
- 其他费用：XX万
- **合计**：XX万

**总产出**：
- 新增贷款额度：XX万
- 年节省利息：XX万
- 其他收益：[如有]

**ROI计算**：
```
方式1：按额度计算
ROI = (新增额度 / 投入成本) × 100% = XX%

方式2：按利息节省计算
年节省利息：XX万
3年节省：XX万
ROI = (3年节省 / 投入成本) × 100% = XX%

方式3：综合计算
考虑额度提升+利率降低，综合ROI = XX%
```

**填写示例**：
```
总投入：
- H业务服务费：6万
- 税费成本：4.5万（增值税及附加）
- 其他费用：0.5万（评估费等）
- 合计：11万

总产出：
- 新增贷款额度：180万（从80万到260万）
- 年节省利息：原80万×12%=9.6万，现260万×3.8%=9.88万
  （注：虽然总利息略增，但考虑到额度增加225%，单位成本大幅下降）
- 利率实际下降：从12%降至3.8%，降幅68%

ROI计算：
方式1：按额度计算
ROI = (180万 / 11万) × 100% = 1636%

方式2：按单位成本下降计算
原贷款成本率：9.6万/80万 = 12%
新贷款成本率：9.88万/260万 = 3.8%
成本率下降：68%

综合评价：ROI极高，投入11万撬动180万额度，且利率大幅下降
```
